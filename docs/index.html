SolarSupplyAu

Visualising daily solar supply

Introduction

SolarSupplyAu calculates the estimated solar supply for almost all postcodes in Australia each day using the most up-to-date installation and weather data.

Modelling Solar Supply

Actual performance data has been analysed from 2,525 different systems Australia-wide. The performace of these systems was compared to the daily weather conditions and combined with machine learning techniques to extrapolate an average performance function. This function can be applied to all installed sytems, not just the ones which provide daily readings.

Applying the Model

With up-to-date information on the solar capacity in each postcode, we can determine the total daily solar supply from the previous day's weather reports.

Visualising Daily Solar Supply

Web mapping services allow us to show detailed 3D visualisation of any part of the country down to the suburb or street level. We can visualise the percentage of residences with solar panels installed, as well as show the daily solar supply by suburb.
